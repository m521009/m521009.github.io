2020-07-30 GitHub GitHub+jsDelivr jsDelivr GitHub图床

今天弄个hexo静态博客，部署在了GitHub上，十年，不知道我会不会一直更新。
还弄了个GitHub图床，https://momotu.cool 用的jsDelivr全球加速，有国内节点，真的美滋滋。

安装教程
推荐使用宝塔安装,建议不要使用虚拟主机安装（估计你也装不上），PHP版本7.2及以上，将整包下载解压（或者直接clone），设置运行目录为/Public,然后配置伪静态，Apache无需配置，nginx伪静态规则在程序根目录有个nginx.txt，复制规则配置即可，之后给所有目录777权限，然后配置/App/Database.php内的数据库信息，最后导入install.sql数据库即可，后台登陆地址/admin，帐号admin，密码123456。可以直接后台更改密码。
